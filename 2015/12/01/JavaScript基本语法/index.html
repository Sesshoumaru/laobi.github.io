JavaScript基本语法

发表于 2015-12-01 | 分类于 JavaScript语法

JavaScript是一种解释型脚本语言，最早之前JavaScript是伴随着html而产生的，浏览器就充当了解析器的角色。后面nodejs的出现则将JavaScript带到了服务器端，nodejs也充当了解析器的角色。不管是浏览器还是nodejs，他们都是基于JavaScript这门语言的语法进行解析的，本文讨论的就是JavaScript语言的基本语法

语句以分号结束

JavaScript并不强制要求在每个语句的结尾加; ，解析器引擎会自动在每个语句的结尾补上;，但是自动加;的时候某些情况下会改变程序的语义，导致结果与预期不符，所以推荐每个语句后面均加上;。

语句块是一组语句集合，用{...}

{...}是可以嵌套的

注释分单行注释和多行注释，注释是给开发人员看到，JavaScript引擎会自动忽略

  • 单行注释以//开头,至行尾的都当做注释
  • 多行注释以/*开头,以*/结尾,之间的内容被视为一大块注释

JavaScript 变量可用于存放值（比如 x=2）和表达式（比如 z=x+y）。

变量的命名

  • 变量必须以字母开头
  • 变量也能以 $ 和 _ 符号开头（不过我们不推荐这么做）
  • 变量名称对大小写敏感（y 和 Y 是不同的变量）

JavaScript是弱语言，同一变量可以存储不同类型的数据
