推荐一个提高python技能的网站

发表于 2017-11-06 | 分类于 python学习

推荐一个提高python技能的网站

在自学的过程中，是否有这样的一种经历？在看了一段时间的python基础之后，不知道如何进一步深入学习？或者说不清楚自己对这些基础知识到底掌握到了哪个层次？

在我看来其实关键的是缺乏实践，任何编程语言都不是简单通过看书就能学会的。这跟学车一个道理，你先得通过理论考试，这个好比看基础书的过程，但是理论考试过了还不能给你颁发驾照，你得去参加实践练车，考科目二、科目三等等。

今天就在这里跟大家分享1个可以实践，用于提高python编程技能水平的网站。网站的名称是https://py.checkio.org。在这里学习者可以通过游戏的方式完成挑战来达到学习编程的目的。网站将实践任务按难以等级进行划分，分别在不同的关卡中，挑战者需要完成关卡内的实践任务进行通关，通关后又能解锁更多、更有难度的关卡。这样的设计让学习有了曲线，由简入繁，非常的适合初学者。再者寓教于乐，让学习者也不那么枯燥。解题的过程，也是一个个的挑战，让学习者也有成就感。所以我想把这个网站推荐给大家，希望能有所帮助。

python是一门简洁而优雅的语言，写出的代码不止能解决问题，而且还要足够的简洁优雅，这也是一个pythoner应有的追求。在这方面这个网站也做的很好，那就是这个网站在你完成了解题后，会开启代码阅读对比功能，你可以查看此道题目所有的其它学习者的解题代码，解题代码按照投票多少的顺序排列，这样你就学习到更多的解题方法，也可以对比学习下自己的代码和其它学习者代码的区别，找出差距，提升自己。

先来看1道题

题目: most-wanted-letter

描述: 给你一个其中包含不同的英文字母和标点符号的文本，你要找到其中出现最多的字母，返回的字母必须是小写形式，当检查最想要的字母时，不区分大小写，所以在你的搜索中 “A” == “a”。 请确保你不计算标点符号，数字和空格，只计算字母。如果你找到 两个或两个以上的具有相同的频率的字母， 返回那个先出现在字母表中的字母。 例如 – “one”包含“o”，“n”，“e”每个字母一次，因此我们选择“e”。

输入: 用于分析的文本.

输出: 最常见的字母的小写形式。

范例:

如何使用: 对于大多数的解密任务，你需要知道各种字母出现在一段文字的频率。例如：如果我们知道在哪个字母出现的频率，我们可以很容易地破解一个简单的加法密码或替换密码。这是语言专家有趣的事情！

前提: 密码只包含ASCII码符号 0 < len(text) ≤ 105

解题代码框架:

网站中每个实践题目，都会给出题目的标题、难易等级、问题的描述、输入参数、输出的结果，还有一个已经提前预设好的解题框架代码。代码里面已经定义好了需要实现功能的函数，和几个测试的案列。你只需要将函数功能实现，然后运行程序，测试下所有的测试用例是否都能通过，如果可以通过，则表明挑战成功。然后将自己的解题代码复制到网站提供的WEBIDE中，run并check,再提交。

对于以上的那道题目，大家有想到解题方法了吗？在我看来至少有4种解题方法。

1 完全面向过程式

2 列表推导式 + ascii_lowercase + min

3 列表推导式 + ascii_lowercase + max

4 列表推导式 + min

感觉还不错的小伙伴，觉得有必要的小伙伴，赶紧去这个网站体验下吧：https://py.checkio.org

这个是一个境外的网站，会有点慢，而且只有少部分的题目提供了中文描述，大家要有耐心哟！这里还向大家推荐下我在GitHub上的一个checkio-python仓库，里面是我个人的在checkio上解题集合，有兴趣的可以star和fork。
