Flask系列1——初步认知

发表于 2017-11-18 | 分类于 Flask系列教程

flask是一款非常流行的Python Web微框架，flask自2010年发布第一个版本以来，大受欢迎，深得开发者的喜爱，并且在多个公司已经得到了应用。今天就来开始通过使用pycharm创建第一个flask程序开认识下flask框架。

创建虚拟环境并使用虚拟环境创建Flask应用程序

为了避免包的混乱和版本的冲突，我们一般都会为每个程序创建虚拟环境，使用Flask我们也不例外。这里我就以pycharm为例，演示下如果创建虚拟环境变量。

  • 打开pycharm，在New Project中我们选择Flask，选择好项目路径后，点击解释器后面的设置按钮，在弹出的的菜单中我们选择Create VirtualEvn。

  • 为创建的虚拟环境设置名称并选择虚拟环境的路径

  • 下拉选择刚刚创建的虚拟环境为项目使用的python解释器

  • 点击create按钮，pycharm就会为我们自动创建一个Flask程序

运行Flask程序

在pycharm中，运行Falsk程序的方式有3种：

  • 点击右上角的绿色三角形按钮
  • 右键点击py文件，在弹出的右键菜单中，选择run’create-falsk’
  • 使用组合快捷键”Ctrl+Shift+F10”

程序启动后，pycharm的run面板会显示程序的启动信息，通过查看信息我们可以看到，flask已经成功运行在了本机的5000端口上

在浏览器中我们输入http://127.0.0.1:5000/地址，回车访问，既可以看到flask的响应信息了

Flask程序初步认识

pycharm的flask模板生成的几句简单的代码，直接就能启动运行，并在浏览器中展示响应的结果，这个是怎么做到的呢？

首先from flask import Flask语句导入了 Flask 类。这个类的实例将会是我们的 WSGI 应用程序，它就是web应用程序中专门负责服务器和客户端交互的。

然后app = Flask(__name__)语句实例化了一个Flask类的实例，这个类的实例也就是WSGI应用程序实例。

然后程序定义了一个视图函数，这个视图函数的返回值是Hello World!的字符串，表示服务器将会把Hello World!作为响应内容发回给浏览器。这个视图函数使用了一个route(/)装饰器进行装饰,这个装饰器的含义就是表名只有访问这个应用程序的根地址时，才会触发视图函数做出响应。

最后我们用 run() 函数来让应用运行在本地服务器上。 其中 if __name__ == '__main__': 确保服务器只会在该脚本被 Python 解释器直接执行的时候才会运行，而不是作为模块导入的时候。

以上就是一个最基本的Flask应用，我们后面将逐步扩散讲解Flask应用的功能点。
