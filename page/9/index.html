JavaScript运算符

发表于 2015-12-21 | 分类于 JavaScript语法

JavaScript 具有丰富多样的运算符，包括算术运算符、逻辑运算符、按位运算符、赋值运算符以及一些其他运算符。

算术运算符(加法+、减法-、乘法*、除法/、取模%、自增++、自减–、取反-)

加法 result = expression1 + expression2;

将一个数值表达式的值与另一个数值表达式的值相加，或连接两个字符串。

  • 两个表达式都是字符串,则将2个字符串连接成1个字符串

  • 其中一个表达式为字符串，而另一个表达式为非字符串,将非字符串表达式转换成字符串后再与另外一个字符串连接成结果字符串

  • 两个表达式都为非字符串,则按数学加法相加(true转换成1,false转换成0,null转换成0,undefined转换成NaN)。

null表示一个空的值，它和0以及空字符串’’不同，0是一个数值，’’表示长度为0的字符串，而null表示“空”。

它与c#语言中的null，Python中过的None类似。

注意运算符typeof对null的求值结果是object,而不是null

undefined表示对象属性不存在、或已声明但未赋值的变量

JavaScript基本语法

发表于 2015-12-01 | 分类于 JavaScript语法

JavaScript是一种解释型脚本语言，最早之前JavaScript是伴随着html而产生的，浏览器就充当了解析器的角色。后面nodejs的出现则将JavaScript带到了服务器端，nodejs也充当了解析器的角色。不管是浏览器还是nodejs，他们都是基于JavaScript这门语言的语法进行解析的，本文讨论的就是JavaScript语言的基本语法

语句以分号结束

JavaScript并不强制要求在每个语句的结尾加; ，解析器引擎会自动在每个语句的结尾补上;，但是自动加;的时候某些情况下会改变程序的语义，导致结果与预期不符，所以推荐每个语句后面均加上;。

语句块是一组语句集合，用{...}

{...}是可以嵌套的

注释分单行注释和多行注释，注释是给开发人员看到，JavaScript引擎会自动忽略

  • 单行注释以//开头,至行尾的都当做注释
  • 多行注释以/*开头,以*/结尾,之间的内容被视为一大块注释
